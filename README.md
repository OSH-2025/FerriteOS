|   选题   |    3月23日     | 第三次线下讨论。老师否认了基于Rust的WebAssembly解释器和运行时，同时我们对于嵌入式开发了解较少，放弃了关于freertos以及wasm虚拟机在嵌入式系统的应用等选题。经过和老师的进一步讨论，我们重新确立选题为使用Rust重构LiteOS部分模块。 |
